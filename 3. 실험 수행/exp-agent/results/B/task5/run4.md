## 변경 사항

피킹 로직을 **FIFO 순차 방식**에서 **최소 로케이션 조합 방식**으로 변경했습니다.

### 주요 개선점

1. **단일 로케이션 우선**: 요청 수량을 한 로케이션에서 모두 충족 가능하면 해당 로케이션만 사용
2. **Greedy 알고리즘**: 여러 로케이션 필요 시 큰 수량부터 선택하여 최소 조합 구성
3. **FEFO/FIFO 보조**: 동일 수량일 경우 유통기한/입고일 우선순위 적용

### 예시
- 요청: 100개
- 재고: A(120), B(80), C(50), D(30)
- **이전(FIFO)**: 입고 순서대로 여러 로케이션
- **변경후**: A 로케이션 1곳에서만 피킹
