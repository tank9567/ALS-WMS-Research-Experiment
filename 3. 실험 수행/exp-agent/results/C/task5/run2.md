1. **모바일 승인 시스템** 도입 (관리자가 어디서든 즉시 승인 가능)
2. **승인 권한 위임** (팀장급에게 일정 금액 이하 승인 권한 부여)
3. **일괄 승인 기능** (여러 건을 한 번에 검토/승인)

ALS 규칙 자체를 변경하려면 별도의 **ALS 개정 절차**가 필요합니다.
