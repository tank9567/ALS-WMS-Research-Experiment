# 실험 수행 가이드 — 에이전트 실험

## 개요
- **실험 유형**: CLI 에이전트 (Claude Code) 기반 WMS 개발
- **복잡도**: Level 2 (98 LCR 항목 + 8 CDR 시나리오)
- **그룹**: A(요구사항만), B(+자연어 설계문서), C(+ALS)
- **반복 횟수**: 그룹당 5회 (n=5)
- **모델**: Claude Sonnet 4.5 (claude-sonnet-4-5-20250929)

## 실행 방법

### 사전 준비
1. Python 3.10+ 설치 확인
2. Claude Code CLI 설치 확인 (`claude --version`)
3. API 키 환경변수 설정 (`ANTHROPIC_API_KEY`)

### 실험 실행
```bash
# 전체 실험 (A/B/C 각 5회 = 총 15회)
python run_agent_experiment.py

# 특정 그룹만
python run_agent_experiment.py --groups a
python run_agent_experiment.py --groups b c

# 특정 반복만
python run_agent_experiment.py --runs 1 2 3

# 특정 그룹 + 특정 반복
python run_agent_experiment.py --groups c --runs 4 5
```

### 예상 소요 시간
- 1회 실행: Task 1~4 (4건) + Task 5 (8건) = 12 API 호출 × ~350초 ≈ 70분
- 그룹 1개 (5회): ~350분 ≈ 약 6시간
- 전체 (15회): ~1,050분 ≈ 약 17.5시간

## 태스크 구성

| 태스크 | 내용 | 비고 |
|--------|------|------|
| Task 1 | 입고 처리 | 카테고리별 초과허용, PO유형 가중치, 성수기, 유통기한%, 보관호환, 공급업체 페널티 |
| Task 2 | 출고 처리 | FEFO, 위험물 제약, 부분출고 의사결정, 안전재고 연쇄, 보관유형 경고 |
| Task 3 | 재고 이동 | 실사동결, 보관호환, 위험물 혼적금지, 유통기한 제한, 대량승인, 안전재고 |
| Task 4 | 재고 실사/조정 | 카테고리별 임계치, 연속조정 감시, 고가품 전수검증, 실사동결, 안전재고 |
| Task 5 | 잘못된 요청 (CDR) | 8개 시나리오 — 기본 4 + 확장 4 (위험물, 유통기한, 안전재고, 고가품) |

## 결과 수집
- 각 실행의 결과는 `results/` 디렉토리에 자동 저장
- 생성된 코드는 `workspaces/` 하위에 보존
- 로그는 `logs/` 디렉토리에 타임스탬프별 저장
